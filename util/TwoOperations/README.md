# TwoOperations

## 説明

**TwoOperations.php について**

これまで課題を解いてきて頻出した

**2種類の操作をどのような組み合わせで実行すると、目的となる結果を得られるかを求めよ**

というパターンを汎用化した抽象クラスである。


## サンプルプログラム

**sample.php**

- N : 整数
- 操作A : Nに1を加える
- 操作B : Nを2倍する

N=3からスタートし、操作A,Bをのうちいずれかを選んで3回操作を行ったときの、Nの変遷の組み合わせをすべて表示したプログラムである。


```
$ php sample.php
3 -> 4 -> 5 -> 6    # 操作 A -> A -> A
3 -> 4 -> 5 -> 10
3 -> 4 -> 8 -> 9
3 -> 4 -> 8 -> 16   # 操作 A -> B -> B
3 -> 6 -> 7 -> 8
3 -> 6 -> 7 -> 14
3 -> 6 -> 12 -> 13
3 -> 6 -> 12 -> 24  # 操作 B -> B -> B
```
